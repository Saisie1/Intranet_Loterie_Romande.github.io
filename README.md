# Intranet_Loterie_Romande
Site HTML, CSS et JavaScript. Intranet pour la loterie romande. Ceci est un TE au CPNE qui consiste a faire un intranet pour l'entreprise de notre choix et j'y serait donc noter

If the loading bar is bothering you, just do CTRL + F on the Index.HTML and search for "loader time" and set the value number to 0 and save then it won't show up again.

This site has some default values on the news and calendar if you don't have them anymore and wanna get them back just right click on the page and press "inspect" then go on console and type "localstorage.removeitem('newsData')" then enter. When you will reload the page they will come back here are the 3 principal ones

"localstorage.removeitem('newsData')" == reset the default "Actualit√©es"
"localstorage.removeitem('eventsData')" == reset the default calendar "Visites"
"localstorage.removeitem('userName')" == reset the username and next F5 it will ask the name again
